## Env Setup
- Install `python`
  - Install `poetry`
- Install `npm`

## QoL
- In the backend directory:
  - `poetry run python manage.py createsuperuser`

## Running
- In the backend directory:
  - `poetry run python manage.py runserver`
- In the frontend directory:
  - `yarn start`
